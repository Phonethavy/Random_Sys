## ЁЯУЭ API Endpoints

### Participants
- `POST /api/participants/upload` - Upload Excel file
- `GET /api/participants` - Get all participants
- `GET /api/participants/eligible` - Get eligible participants

### Draw
- `POST /api/draw/conduct` - Conduct lucky draw
- `GET /api/draw/history` - Get draw history
- `GET /api/draw/winners` - Get all winners

### Export
- `GET /api/export/eligible/excel` - Export eligible to Excel
- `GET /api/export/eligible/pdf` - Export eligible to PDF
- `GET /api/export/winners/excel` - Export winners to Excel
